# Shared Prompt Rules

Use `docs/reference-packs/document-reference-policy.md`.

Every Phase 6.5 packet must state:
- work package ID and exact branch;
- mandatory reads before optional references;
- allowed paths from `docs/spec-registry/task-to-file-contract-map.yaml`;
- region marker/operation mode;
- acceptance criteria copied from the work package registry;
- commands/evidence required for review.

Do not reference long source docs unless the Human Lead explicitly marks a trace ambiguity.
